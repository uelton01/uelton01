# 👋 Sobre mim

Sou Tecnólogo em Gestão Ambiental e fundador da **Arkhevida**, uma iniciativa dedicada ao desenvolvimento de soluções que unem Inteligência Artificial, Ciência e Sustentabilidade.

Atualmente desenvolvo o **SASE (Scientific Autonomous Semantic Engine)**, uma plataforma de Inteligência Artificial criada para interpretar documentos científicos, construir relações semânticas e apoiar a tomada de decisão baseada em evidências.

Meu objetivo é desenvolver tecnologias que contribuam para uma gestão ambiental mais inteligente, transparente e sustentável.

# 🚀 Projeto Principal

## 🧠 SASE — Scientific Autonomous Semantic Engine

O SASE é uma plataforma de Inteligência Artificial voltada para:

- 📄 Interpretação de documentos científicos
- 🧠 Raciocínio semântico
- 🕸️ Construção de grafos de conhecimento
- 📊 Apoio à tomada de decisão
- 💡 Inteligência Artificial Explicável (Explainable AI)
- 🔍 Recuperação inteligente de informações (RAG)
